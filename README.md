# Tic-Tac-Toe Game with React

Este es un juego de Tic-Tac-Toe desarrollado con React, que permite jugar múltiples juegos y llevar un registro de los juegos ganados. Además, el juego cuenta con un marcador, permite resetear la partida en cualquier momento y también permite avanzar a la siguiente ronda. La interfaz del juego es responsive, por lo que se adapta a diferentes tamaños de pantalla.

## Características

- Jugar múltiples juegos de Tic-Tac-Toe
- Llevar un registro de los juegos ganados
- Marcador para llevar la cuenta de los juegos ganados
- Guardado de registros en el Local Storage del navegador
- Interfaz responsive que se adapta a diferentes tamaños de pantalla
- Reiniciar el juego en cualquier momento
- Ir a la siguiente ronda del juego

## Tecnologías Utilizadas

- HTML
- CSS
- Javascript
- React

## Uso

Para jugar al juego, sigue estos pasos:

1. Clona el repositorio en tu máquina local: 

```
git clone https://github.com/tu_usuario/tic-tac-toe-react.git
```

2. Instala las dependencias:

```
npm install
```

3. Inicia el servidor local:

```
npm start
```

4. Abre el juego en tu navegador en la dirección `http://localhost:3000`.

5. Juega al Tic-Tac-Toe y diviértete!
